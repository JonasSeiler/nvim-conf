# My nvim config 


## current plugins:
### plugin Manager
- [lazy-nvim](https://github.com/folke/lazy.nvim): the plugin manager of my choice 
### LSP
- [nvim-cmp](https://github.com/hrsh7th/nvim-cmp): lsp autocompletion tool 
- [lsp_signature-nvim](https://github.com/ray-x/lsp_signature.nvim) shows function signatures 
- [nvim-lspconfig](https://github.com/neovim/nvim-lspconfig) LSP server config collection
- [mason-nvim](https://github.com/mason-org/mason.nvim) LSP package manager 
### functional
- [telescope.nvim](https://github.com/ThePrimeagen/harpoon) fuzzy finder  
- [harpoon](https://github.com/ThePrimeagen/harpoon) allows you to jump between frequently used files via keybinds 
- [undotree](https://github.com/mbbill/undotree) allows going through the tree of changes in a file 
- [neogen](https://github.com/danymat/neogen) documentation generation tool 
- [nvim-treesitter](https://github.com/nvim-treesitter/nvim-treesitter) syntax parser generator 
- [vim.fugitive](https://github.com/tpope/vim-fugitive) git wrapper, (only installed not using it right now but should start to)
### visuals
- [Lualine.nvim](https://github.com/nvim-lualine/lualine.nvim) a pretty status bar 
- [transparent.nvim](https://github.com/xiyaowong/transparent.nvim): allows for transparent backgrounds 
- [monokai-pro](https://github.com/loctvl842/monokai-pro.nvim): current nvim theme (i changed basically every color) 
- [rosepine/nvim](https://github.com/rose-pine/neovim): currently not used but installed
